# 👥 Sistema de Cadastro de Pessoas em Python

Um sistema interativo em linha de comando (CLI) em Python que permite cadastrar e visualizar pessoas, verificando se o arquivo de armazenamento existe e criando-o automaticamente caso necessário.

## 📌 Funcionalidades

* 📜 **Visualizar Cadastro:** Exibe a lista completa das pessoas cadastradas com nome e idade.
* ➕ **Novo Cadastro:** Permite registrar novas pessoas no sistema.
* 🛡️ **Verificação Automática:** Cria o arquivo de dados automaticamente (`cadastros.json`) caso ele ainda não exista na pasta.
* ⚠️ **Tratamento de Erros:** Validação de entradas para garantir que a idade informada seja um número válido.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **JSON** (para armazenamento dos cadastros)
