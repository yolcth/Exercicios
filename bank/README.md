# 🏦 Sistema Bancário Simples em Python

Um sistema de simulação bancária em linha de comando (CLI) feito em Python, com persistência de dados em arquivos JSON.

## 📌 Funcionalidades

* 👤 **Cadastro e Login:** Sistema de autenticação com validação de idade (maiores de 18 anos).
* 💰 **Consulta de Saldo:** Exibição do saldo atual do usuário.
* 💵 **Depósito:** Entrada de dinheiro na conta com atualização em tempo real.
* 💸 **Saque:** Retirada de dinheiro com verificação de saldo disponível.
* 🔄 **Transferência:** Envio de valores entre contas.
* 📁 **Persistência de Dados:** Todos os cadastros e saldos ficam salvos em um arquivo `historico.json`.

## 🛠️ Tecnologias Utilizadas
    
* **Python 3**
* **JSON** (para armazenamento dos dados)
