# 📊 Sistema de Notas de Alunos em Python

Um sistema em linha de comando (CLI) desenvolvido em Python para registro e acompanhamento de alunos e suas notas escolares, com persistência de dados em formato JSON.

## 📌 Funcionalidades

* 📜 **Ver Alunos:** Lista os alunos cadastrados no sistema junto com suas notas.
* ➕ **Cadastrar Novo Aluno:** Registra um aluno com 4 notas para acompanhamento.
* 💾 **Persistência de Dados:** Salva e carrega os registros do arquivo `notas.json`.
* ⚠️ **Tratamento de Erros:** Validação de dados de entrada para evitar falhas com digitações incorretas.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **JSON** (para armazenamento dos dados)
