# 📚 Exercícios

Repositório com os exercícios e projetos práticos que venho desenvolvendo enquanto aprendo programação.

## 👋 Sobre

Tenho 15 anos e estou construindo uma base sólida em programação, com foco em Python, backend, boas práticas e arquitetura de software. Em breve começo a estudar Java. Esse repositório é onde guardo os exercícios e pequenos projetos que uso pra praticar o que vou aprendendo, não é código de produção, é material de estudo e evolução constante.
Cada pasta tem seu próprio `readme.md` com mais detalhes sobre o projeto específico.

## 🎯 Objetivo

Não é chegar rápido em lugar nenhum, é evoluir de forma consistente, entendendo os conceitos em profundidade em vez de só decorar código.
